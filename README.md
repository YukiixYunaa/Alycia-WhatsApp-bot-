Bot WhatsApp yang dibuat dengan NodeJS dan Termux.

## Cara Menjalankan
1. Clone repo: `git clone https://github.com/username/pengingat-bot.git`
2. Install dependencies: `npm install`
3. Jalankan bot: `node index.js`
4. Scan QR WhatsApp Web (jika perlu)

## Catatan
- Jangan commit folder `session` atau `node_modules`

cara install :
- cd ~/pengingat-bot
- npm init -y
- npm install qrcode-terminal
- npm i node-cron node-fetch
- npm install node-fetch
- npm i node-fetch@2
- pkill node
node index.js
- nano index.js (masukin script index js)
- npm start
- scan kode qr
- jika sudah terhubung maka tes dengan command .ping jika berhasil bot akan membalas pong
- sekian gomenne kalo masih ada error nya
